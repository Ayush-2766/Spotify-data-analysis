Spotify Song Popularity Analysis

📌 Overview

Analysis of Spotify song data to explore how audio features relate to popularity.

⚙️ Process

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation heatmap and visualizations

🔍 Key Insights

- No feature shows strong correlation with popularity
- Danceability and loudness show slight positive trends
- Acoustic songs tend to be less popular
- Popularity depends on multiple factors rather than a single feature

🛠 Tools

Python, Pandas, Matplotlib, Seaborn
